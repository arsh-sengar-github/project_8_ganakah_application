<!-- ABOUT -->

## About

![Ganakah][ganakah-url]

Ganakah applies machine learning techniques to procurement and inventory datasets in order to extract meaningful insights and detect potential anomalies.

The application is designed as a lightweight analytical tool that allows users to interact with trained models through a simple web interface.

---

<!-- IDEA -->

### Idea

The goal of Ganakah is to demonstrate how machine learning models can assist in procurement intelligence by:

- estimating logistics costs
- identifying suspicious or anomalous invoice records
- enabling quick analytical insights through a web application

## Use

Ganakah provides two primary capabilities:

| Capability              | Description                                                                                                                  |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| Freight Cost Prediction | Users can input procurement values and obtain predicted freight costs based on trained regression models.                    |
| Invoice Risk Detection  | Users can provide invoice and purchase order details to determine whether the invoice may be potentially risky or anomalous. |

<!-- INSTALL -->

### Install

Create virtual environment:

```bash
python -m venv .venv
```

Activate virtual environment:

Windows:

```bash
.venv\Scripts\activate
```

Linx / macOS:

```bash
source .venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run application:

```bash
streamlit run app.py
```
